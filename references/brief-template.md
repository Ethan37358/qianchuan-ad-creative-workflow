# Product Brief Template

Use this template when the user wants a repeatable Qianchuan creative workflow.

## Minimum Brief

```text
商品/类目：
价格/优惠：
投放目标：直播间引流 / 商品卡成交 / 线索 / 复购召回
目标人群：
核心卖点：
真实证据：
使用场景：
已有素材：
禁用表达：
参考竞品/参考素材：
期望风格：
视频时长：15s / 30s / 45s
```

## Clarify Only High-Risk Gaps

Ask follow-up questions when missing data could cause false claims or unusable creative:

- Price, offer, inventory, or live-room mechanism is unclear.
- The product requires regulated claims: health, finance, medical, education, cosmetics efficacy.
- The user asks for comparison but provides no proof.
- The user asks for user feedback but provides no real review material.
- The user wants "best", "first", "guaranteed", "cure", "100%" or similar absolute claims.

Otherwise proceed with assumptions and label them.
