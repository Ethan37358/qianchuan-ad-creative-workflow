# Material Workflow

Use this when producing素材需求,素材采集, or draft video assembly plans.

## Source Priority

Prefer materials in this order:

1. Real product footage
2. Livestream clips
3. User-generated review/demo footage with permission
4. Brand-owned images and videos
5. Re-shot simple demo scenes
6. Stock footage for background context only
7. AI-generated visuals when product truth is not being represented as real footage

For Qianchuan conversion ads, stock or AI footage should not replace real product demonstration when the selling point depends on product performance.

## Material Modes

### Local Material Mode

Use when the user provides a folder.

Steps:

1. Inventory videos/images by folder and filename.
2. Identify likely intent from folder names, such as 材质安全, 立减, 免费试睡, 信息流, 直播切片, 商品图.
3. Generate contact sheets or frame previews where possible.
4. Check orientation, duration, and resolution.
5. Bind scene candidates with首选 and备选.
6. Mark uncertainty when visual content has not been fully reviewed.

### Network Acquisition Mode

Use when the user has no local material or asks to补充素材.

Steps:

1. Generate Chinese and English search terms per scene.
2. Separate product-proof scenes from background/context scenes.
3. Search or prepare candidates from approved stock/video sources when tools and rights allow it.
4. Download or list candidates with source URLs and license notes when available.
5. Never use stock footage as proof for product effects, materials, certifications, or真实反馈.

Good network素材 uses:

- Office fatigue background
- Bedroom/lifestyle atmosphere
- Generic sleep routine
- Abstract premium texture
- Hand/desk/phone transition shots

Bad network素材 uses:

- Fake product demonstration
- Fake before/after
- Fake buyer review
- Fake certification/testing footage
- Product-specific claims not belonging to the product

### Hybrid Mode

Default for many real projects:

```text
local product footage + local structure/material proof + network background/context补充
```

Use network footage to improve pacing and atmosphere, not to replace product truth.

## Material Matching Rules

- The first 3 seconds need high relevance, product visibility, or strong scenario tension.
- Every claimed selling point needs matching visual proof or a lower-risk wording.
- If using a livestream clip, mark the exact spoken point or product demonstration needed.
- If using review screenshots, avoid fabricating reviews and avoid exposing private user data.
- If using stock footage, label it as background/context, not proof.
- If using AI visuals, avoid showing impossible product functions or fake before/after results.

## Candidate Scoring

Score candidate footage from 1-5:

- Relevance to口播
- Product visibility
- Clarity/resolution
- Trustworthiness
- Platform-native feel
- Compliance safety
- Edit flexibility

Recommend human approval for any clip under 4 on trustworthiness or compliance safety.
